# vim
vimrc config file without plugins
# Config & Usage

```ruby
cd ~ ; git clone https://github.com/notwidow/vim ; mv vim .vim ; cd .vim/ ; mv .vimrc ~/. ; cd
```
# Features
> No plugins      
> One file to rule it all         
> Builtin nightfly colorscheme       
> Colorful statusline      
> Autopairs
